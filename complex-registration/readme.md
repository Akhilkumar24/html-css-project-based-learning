# 🚀 Advanced Student Registration Form (HTML Only)

A **feature-rich and structured Student Registration Form** built using **pure HTML**. This project focuses on mastering form design, input handling, and semantic HTML—ideal for beginners preparing for web development interviews and portfolio building.

---

## 📌 Project Overview

This project demonstrates how to build a **complex registration form using only HTML**, without any CSS or JavaScript. It includes a wide range of input types and form elements to simulate a real-world student registration system.

---

## ✨ Features

### 📝 Comprehensive Form Fields

* Full Name (First Name, Last Name)
* Email Address
* Password & Confirm Password
* Phone Number
* Date of Birth
* Gender Selection (Radio Buttons)
* Address (Textarea)
* City, State, Country
* Pincode / ZIP Code
* Course Selection (Dropdown)
* Skills Selection (Checkboxes)
* Upload Profile Photo
* Upload Documents

---

### ✅ Built-in HTML Validations

* `required` fields
* Email validation (`type="email"`)
* Password constraints (`minlength`, `maxlength`)
* Number validation (`type="number"`)
* Date picker (`type="date"`)
* File upload restrictions (`accept` attribute)
* Placeholder guidance

---

### 📂 Form Functionalities

* Organized using `<fieldset>` and `<legend>`
* Proper labeling using `<label>`
* Grouped inputs for better readability
* Submit and Reset buttons
* Accessible and semantic structure
