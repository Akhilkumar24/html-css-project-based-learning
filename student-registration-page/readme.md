# Student Registration Form (HTML Project)

## 📌 Project Overview

This project is a **Student Registration Form built using HTML**.
The purpose of this project is to practice **HTML form elements** and understand how user input is collected in web applications.

The form allows students to enter their personal and academic details such as **name, email, phone number, department, skills, and address**.

---

## 🎯 Learning Objectives

Through this project, I practiced and learned the following concepts:

* Creating **HTML forms**
* Using different **input field types**
* Implementing **radio buttons and checkboxes**
* Creating **dropdown menus using select**
* Adding **textarea for user input**
* Organizing form fields using **labels and line breaks**
* Creating **submit and reset buttons**

---

## 🧰 Technologies Used

* HTML5

---

## ✨ Features

The Student Registration Form includes the following features:

* Full Name input field
* Email input field
* Phone number input field
* Gender selection using radio buttons
* Department selection using dropdown menu
* Skills selection using checkboxes
* Address input using textarea
* Submit button to register
* Reset button to clear the form
