# AI Learning Platform - Landing Page

## 📌 Project Overview

This project is a **modern landing page built using HTML and CSS**.
It represents a simple **AI-powered learning platform** and demonstrates how real-world websites are structured using frontend technologies.

The landing page includes key sections like **navigation bar, hero section, features, about, and footer**, making it similar to real startup websites.

---

## 🎯 Learning Objectives

Through this project, I learned and practiced:

* Structuring a complete webpage using **HTML**
* Styling layouts using **CSS**
* Using **Flexbox** for alignment and layout
* Creating a **responsive-like design structure**
* Designing clean and modern UI components
* Organizing content into reusable sections

---

## 🧰 Technologies Used

* HTML5
* CSS3 (Flexbox)

---

## ✨ Features

* Responsive-style **navigation bar**
* Attractive **hero section with call-to-action button**
* **Features section** with card layout
* Informative **about section**
* Clean and minimal **footer**
* Consistent color theme and spacing
